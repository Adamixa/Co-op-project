# Co-op Project

## Overview

This repository contains the source code for the University Restaurant Information project, a web-based application designed to manage and display information about university restaurants.

## Features

- **Restaurant Management**: Add, update, and remove restaurant details.
- **Menu Display**: Show menus with detailed information.
- **User Authentication**: Secure login and registration system.

## Technology Stack

- **Front-end**: HTML, CSS, JavaScript
- **Back-end**: C#
- **Database**: SQL Server

## Getting Started

### Prerequisites

- [.NET Framework](https://dotnet.microsoft.com/download)
- [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Adamixa/Co-op-project.git

2. Open the solution file UniversityResturantInformation.sln in Visual Studio.

3. Restore NuGet packages and build the solution.

4. Configure the database connection string in appsettings.json.

5. Run the application.

### Usage

- Navigate to the homepage to view restaurant listings.
- Use the admin panel to manage restaurants and menus.
